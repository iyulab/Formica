## 📚 문서 활용 가이드

### 처음 시작하는 경우
1. **개요**와 **왜 문서중심인가?** 먼저 읽기
2. 자신의 역할에 맞는 추천 경로 따라가기
3. 워크숍 준비 후 실습 진행
4. 사례 연구로 실무 감각 익히기

### 특정 주제만 필요한 경우
- 문서 인덱스 테이블에서 필요한 문서 직접 찾기
- 각 문서는 독립적으로 읽을 수 있도록 구성됨

### 팀 교육용으로 사용하는 경우
1. **소개** 섹션을 함께 읽기 (1시간)
2. **이론** 섹션 중점 학습 (1시간 30분)
3. **워크숍 가이드** 검토 후 실제 워크숍 진행 (3시간)
4. **사례 연구** 함께 분석 (1시간)
5. 역할별로 **구현 원리** 섹션 분담 학습
